## Troubleshooting

### "Nothing happens when I import"
- Make sure you selected video or image files
- Check that your browser allows file access
- Try using Chrome or Safari

### "Video preview is black"
- The video format might not be supported
- Try converting to MP4 format
- Refresh the page and try again

### "Can't find my saved draft"
- Make sure you're using the same browser
- Check that cookies/storage is enabled
- Look for the exact name you saved it as

### "Export takes forever"
- This is normal for longer videos
- Wait for the progress bar to complete
- Don't close the browser during export
