## â“ Troubleshooting

### "Command not recognized"
**Cause:** AI couldn't understand the command

**Try:**
- Use simpler words: "slow motion" instead of "decelerate"
- Specify the clip: "clip 2" or "at 0:15"
- Check examples above

**Need help?** Type commands like the examples - AI learns from these patterns!

### "No clips to apply to"
**Cause:** Timeline is empty or clip not found

**Fix:**
- Add clips to timeline first
- Check clip numbers (clip 1, not clip 0)
- Make sure time exists in video

### Command didn't work as expected
**Cause:** AI interpreted differently

**Try:**
- Be more specific: "clip 2" instead of "the middle one"
- Use exact time: "at 0:15" instead of "around 15 seconds"
- Check current effects first
